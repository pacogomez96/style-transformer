# Style Transformer

Transform dense academic writing into Mark Manson's conversational style.

## Features

✨ Real-time transformations  
📱 Mobile optimized (installable on Android)  
🎯 Beautiful UI with dark theme  
🔒 API key stored locally  
⚡ PWA support  

## Quick Start

1. Get API key from https://console.anthropic.com
2. Open the app
3. Enter your API key
4. Paste academic text
5. Click Transform
6. Download or copy results

## Installation on Phone

**Android:**
1. Open in Chrome
2. Tap menu (three dots)
3. Tap "Install app"
4. Done!

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Anthropic Claude API
- PWA
